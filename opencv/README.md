# Real-Time Face Detection using OpenCV

## Description

This project detects human faces in real-time using OpenCV and a Haar Cascade classifier.

## Features

* Real-time webcam face detection
* Draws bounding box around faces
* Displays number of faces detected

## Technologies

* Python 3.10
* OpenCV
* NumPy

## How to Run

pip install -r requirements.txt

python opencv_project.py

## Controls

Press 'q' to exit

## Output

Detects faces and shows them with rectangles in a live camera feed.
