# Real-Time Object Detection using YOLOv8

## Overview

This project performs real-time object detection using YOLOv8 and OpenCV. It captures video from a webcam and detects multiple objects such as people, phones, and more.

## Features

* Real-time object detection
* Uses YOLOv8 deep learning model
* Detects multiple objects simultaneously
* Displays bounding boxes and labels

## Technologies Used

* Python 3.10
* YOLOv8 (Ultralytics)
* OpenCV
* NumPy

## Installation

pip install -r requirements.txt

## Run the Project

python yolo.py

## Controls

Press 'q' to exit

## Output

Displays webcam feed with detected objects and labels.
